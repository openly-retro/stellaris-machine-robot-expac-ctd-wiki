Vanilla Overwrites Mods
#######################

These mods improve things in the **base game**, via overrides, for use with "Machines & Robot Expansion: Continued".

World Machine Origin Vanilla Overwrites
---------------------------------------

**Author**: `Legit Rikk`_

**Steam workshop link**: https://steamcommunity.com/sharedfiles/filedetails/?id=3401666625

**Description** (from the mod workshop page):

	I overwrote the vanilla tradition trees to add world machine specific replacement traditions if you use the origin, so it doesn't feel terrible to select things that boost pop jobs (among other things). Compatible with Gigastructural Engineering, But probably not a lot else that would change the vanilla tradition trees.


Machine & Robot Expansion Ctd: More Trait Points + Picks
--------------------------------------------------------

**Author**: Openly Retro

**Steam Workshop Link**: https://steamcommunity.com/sharedfiles/filedetails/?id=3653979426

**Description**:

	This mod increases trait points and picks for **Machines and Robots only**. It is necessary since the main mod overwrites the MACHINE archetype and this messes with mods that boost point and picks. So a custom mod like this one is necessary.

	This mod does NOT raise trait points and picks for biological species or lithoids.
