Compatibility Mods
##################

A list of sub-mods that provide compatibility between this mod and another.

World Machine Origin Gigastructures Buildings
---------------------------------------------

**Target mod**: Gigastructural Engineering

**Author**: `Legit Rikk`_

**Steam workshop link**: https://steamcommunity.com/sharedfiles/filedetails/?id=3412089966 

**Description** (from the mod workshop page):

	Machine & Robot Expansion: Continued just released a really cool origin about pop-less production! Too bad so many mods rely on pops! I made a patch for Gigastructures that adds copies of buildings that rely on pops... *(continued on mod workshop page)*


World Machine Origin DoA Buildings
----------------------------------

**Target mod**: Dawn of Ascension

**Author**: `OlegY`_

**Steam workshop link**: https://steamcommunity.com/sharedfiles/filedetails/?id=3677086971

**Description** (from the mod workshop page):

	This mod changes Dawn of Ascension buildings to fully automated, pop-less versions for empires with World Machine Origin from Machines & Robot Expansion Continued. Civics Dark Matter Core and Machine Automation were changed too in accordance with their original buffs.


World Machine Origin ACoT Buildings
-----------------------------------

**Target mod**: A Cache of Technologies (ACoT)

**Author**: `OlegY`_

**Steam workshop link**: https://steamcommunity.com/sharedfiles/filedetails/?id=3679636017

**Description** (from the mod workshop page):

	This mod changes ACOT buildings and district zones to fully automated, pop-less versions for empires with World Machine Origin from Machines & Robot Expansion Continued.
